# ⌨️ Keyboard Shortcuts Quick Reference

## 🎯 Quick Access

| Shortcut | Action | Description |
|----------|--------|-------------|
| `?` | Toggle Help | Open/close the help panel |
| `Ctrl + S` | Save | Save all drawings to JSON file |
| `Ctrl + P` | Draw Parking | Start drawing a parking area |
| `Ctrl + R` | Draw Route | Start drawing a route |
| `Ctrl + G` | Draw Gate | Start placing a gate marker |
| `Ctrl + E` | Entry Point | Start placing an entry point |
| `Ctrl + D` | Delete | Delete selected items |
| `Ctrl + H` | Statistics | Toggle statistics dashboard |
| `Ctrl + Shift + R` | Simulation | Toggle real-time simulation |
| `Ctrl + F` | Search | Focus search bar |
| `Esc` | Cancel | Cancel current action/close modals |

## 🗺️ Map Navigation

| Shortcut | Action |
|----------|--------|
| `Arrow Keys` | Pan the map |
| `+` / `-` | Zoom in/out |
| `Ctrl + Drag` | Rotate map (3D view) |
| `Shift + Drag` | Tilt map |
| `Double Click` | Zoom to location |

## 🎨 Drawing Tools

| Shortcut | Tool |
|----------|------|
| `P` | Parking Area mode |
| `R` | Route mode |
| `G` | Gate mode |
| `E` | Entry Point mode |
| `Delete` | Delete selected |
| `Backspace` | Delete selected |

## 💡 Pro Tips

1. **Hold Shift** while drawing to snap to straight lines
2. **Double-click** to finish drawing a polygon
3. **Click + Drag** vertices to edit shapes
4. **Ctrl + Z** to undo (if implemented)
5. **Ctrl + Y` to redo (if implemented)

## 📱 Mobile Gestures

| Gesture | Action |
|---------|--------|
| **Tap** | Select/Click |
| **Double Tap** | Zoom in |
| **Pinch** | Zoom in/out |
| **Two-finger Drag** | Pan the map |
| **Two-finger Rotate** | Rotate map |
| **Long Press** | Context menu |
